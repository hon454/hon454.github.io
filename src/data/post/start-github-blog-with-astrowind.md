AstroWind를 기반으로 `hon454.github.io` 블로그를 시작했다. 목표는 단순하다. 개발하며 배운 내용, 프로젝트를 만들며 남긴 판단, 나중에 다시 참고할 메모를 한곳에 쌓는 것이다.

## 왜 AstroWind인가

AstroWind는 Astro와 Tailwind CSS를 기반으로 만들어진 템플릿이다. 블로그, RSS, SEO 메타데이터, 반응형 레이아웃 같은 기본기가 이미 갖춰져 있어 처음부터 구조를 새로 만들 필요가 적다.

## 이 블로그의 운영 원칙

- 짧은 메모라도 배운 날에 가깝게 남긴다.
- 완성된 결과만이 아니라 선택한 이유와 버린 대안을 함께 적는다.
- 검색해서 다시 찾기 쉬운 제목과 태그를 붙인다.

## 다음 작업

앞으로는 글 분류를 정리하고, 프로젝트 페이지와 이력서 페이지를 추가할 예정이다. 먼저 글을 꾸준히 쌓는 흐름부터 안정화한다.
